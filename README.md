# Ticket Booking System

This is a full-stack ticket booking system built with real booking flow and wallet-based payment logic.  
Users can register, add money to their wallet, view events, reserve seats, confirm bookings, and view booking history.  
Admins can login separately, manage events and seats, monitor bookings, view transactions, and handle refund or cancellation actions.

## Tech Stack

### Frontend

- Vue 3
- TypeScript
- Vue Router
- Axios
- Pinia
- Tailwind CSS

### Backend

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT
- bcrypt
- Zod

## How to Run Backend

Go to the backend folder:

```bash
cd backend
npm install
npm start
```

## How to Run Frontend

Go to the frontend folder:

```bash
cd frontend
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```
